# Environment

## Purpose

Document the runtime environment variables used by the backend.

## Scope

Only variables visible in the current backend source are listed.

## Variables

- `MONGODB_URI`
- `MONGODB_DB_NAME`
- `JWT_SECRET`
- `JWT_EXPIRES_IN`
- `PORT`
- `NODE_ENV`
- `CORS_ORIGIN`
- `SUPER_ADMIN_NAME`
- `SUPER_ADMIN_USERNAME`
- `SUPER_ADMIN_EMAIL`
- `SUPER_ADMIN_MOBILE`
- `SUPER_ADMIN_PASSWORD`
