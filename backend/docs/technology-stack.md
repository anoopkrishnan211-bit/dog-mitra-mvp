# Technology Stack

## Runtime and Frameworks

- **Node.js** - runs the backend server and tooling
- **Express** - HTTP server and routing layer
- **Mongoose** - MongoDB ODM with schema validation and indexes
- **MongoDB Atlas** - managed production database
- **JWT** - stateless authentication for staff sessions
- **bcryptjs** - password hashing
- **zod** - request validation
- **helmet** - security headers
- **cors** - cross-origin access control
- **morgan** - request logging
- **swagger-ui-express** - API documentation UI

## Deployment

- **Render** - hosts the backend API and the separate frontend static site

## Why These Technologies

- Node.js and Express keep the backend lightweight and familiar for API work
- Mongoose gives the project schema enforcement, validation, and indexes
- JWT supports stateless authentication for staff members
- Render provides simple production deployment and environment management
- Swagger UI makes the API discoverable for new developers
