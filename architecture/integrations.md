# Integrations

## Purpose

Document external systems and runtime integrations used by the project.

## Scope

Only integrations that can be confirmed from the repository are included.

## Confirmed Integrations

- MongoDB Atlas
- Render deployment platform
- Swagger UI for API documentation
- JWT token authentication
- WhatsApp and Google Maps links in the UI

## Cross-Application Integration

- Frontend calls backend APIs
- Backend reads and writes MongoDB
- Settings and public content are shared through API responses
