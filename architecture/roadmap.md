# Roadmap

## Purpose

Outline near-term and future evolution of the project.

## Scope

This roadmap is intentionally high level and should be refined as implementation progresses.

## Current Focus

- Stabilize and maintain the existing frontend and backend
- Keep public content editable through the admin path
- Keep MongoDB-backed persistence reliable

## Future Improvements

- Stronger monitoring and observability
- Automated tests
- Rate limiting
- Image upload workflow
- Notification service
- Payment gateway support beyond manual UPI handling
- CI/CD hardening
- API versioning
